# Expense Tracker Application

## Overview
A simple Python-based Expense Tracker application to manage your daily expenses.  
This project allows you to **add, update, and delete expenses** and stores data in a CSV file.

## Features
- Add new expenses with date, category, and amount
- Update or delete existing expenses
- View all expenses in a structured format
- Lightweight and easy to use

## Technologies Used
- Python 3.x
- CSV file for storage
- Command-line interface
